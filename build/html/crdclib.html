crdclib.getCDERecord(cde_id, cde_version=None, verbose=False)

Queries the caDSR API with a CDE identifier and optional version, returns the full JSON object. If no version is given, returns whatever the latest version is.

Parameters:

  • cde_id (Integer) – CDE Public identifier

  • cde_version (String, optional) – The version of the CDE to be queried. If not supplied the latest version will be returned

Returns:

If status_code == 200, a JSON object that is the full CDE record

Return type:

dictionary

Returns:

If status_code != 200, a string with error code and message

Return type:

string

Returns:

If HTTP error, the requests.HTTPError object

Return type:

request.HTTPError
